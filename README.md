Global Suicide Rate Trends and Socioecononic Correlation

Suicide rates vary significantly across regions and demographics, influenced by multiple socioeconomic factors. This project focuses on analyzing global suicide data to identify patterns by age, gender, and country, and to study correlations with economic and social indicators such as GDP per capita and generation cohorts, forecasting future suicide trends and derive actionable insights for prevention programs.
